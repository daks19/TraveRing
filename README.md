## Getting Started

1. Install [Expo CLI](https://docs.expo.dev/get-started/installation/).
2. Clone this repo and run `npm install`.
3. Put your alarm audio in `assets/alarm.mp3`.
4. Run `npx expo start` to open in Expo Go.
